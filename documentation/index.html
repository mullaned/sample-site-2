Setting up Angular Js Version

Angular JS Step One

In app.module.js change the ROOT on line 3 constant value to your website and path the theme is in.

Same goes for the base href in the index.html file on line 4.

app.module.js
app.constant('ROOT', 'http://website.com/SharpVisionV2/angular');
index.html
<base href="/SharpVisionV2/angular/">

Angular JS Step Two

Create an .htaccess file and populate it with the following code.

.htaccess
RewriteCond %{REQUEST_FILENAME} -s [OR] RewriteCond %{REQUEST_FILENAME} -l [OR] RewriteCond %{REQUEST_FILENAME} -d RewriteRule ^.*$ - [NC,L] RewriteRule ^(.*) index.html [NC,L]

Angular JS Step Three

Change the name to whatever you want and your good to go.

app.controllers.js
$scope.brand = 'whatever you want'

Angular JS Top Section

The top section is very simple to user. Just paste in the following into the template you want it to be in.

The sizes by class can be 'large', 'medium' or simply just 'top-parallax'.

app.module.js

Angular JS BOWER AND GRUNT

If you wish to use Grunt and Bower simply move all the contents from the bowergrunt folder into the folder with index.html.

These should include.

bower.json
Gruntfile.js
package.json

More info on both at Grunt and Bower

Grunt runs on Node.js, so if you don't have npm installed already, go ahead and install it. Lets also install git.

In your terminal run:

apt-get install npm
apt-get install nodejs-legacy
apt-get install git

Now lets install Bower and grunt

npm install -g bower
npm install -g grunt-cli

Now lets install the grunt dependencies

npm install grunt
npm install grunt-contrib-concat
npm install grunt-contrib-less
npm install grunt-contrib-uglify
npm install grunt-contrib-watch

And finally lets run bower

bower install

Run the following to activate grunt

grunt

Now in the index.html you can replace all the scripts except the google maps api scripts and the app scripts, to this:

<script src="assets/build/scripts.js"></script>

Same goes for styles, except bootstrap.

<link href="assets/build/styles.min.css" rel="stylesheet">

Angular JS Notes

Because of conflicting code between angular-flexslider and ui bootstrap we have to do the following.

angular-flexslider.js on line 15 match = (attr.slide || attr.flexSlide).match(/^\s*(.+)\s+in\s+(.*?)(?:\s+track\s+by\s+(.+?))?\s*$/);
replace with match = (attr.fslide || attr.flexSlide).match(/^\s*(.+)\s+in\s+(.*?)(?:\s+track\s+by\s+(.+?))?\s*$/);
